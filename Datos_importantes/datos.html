1. PHP tiene mas de 1000 funciones.
  2. $_GET es una matriz de variables pasadas al script actual a través de los parámetros de URL.
  3. $_POST es una matriz de variables pasadas al script actual a través del método HTTP POST.
  4. La información enviada desde un formulario con el método GET es visible para todos (todos los nombres y valores de las variables se muestran en la URL).
  5. GET también limita la cantidad de información que se puede enviar, con un máximo de 2000 caracteres
  6. ¡GET NUNCA debe utilizarse para enviar contraseñas u otra información confidencial!
  7. La información enviada desde un formulario con el método POST es invisible para otros (todos los nombres/valores están integrados dentro del cuerpo de la solicitud HTTP) y no tiene límites en la cantidad de información a enviar.
  8. POST admite funciones avanzadas, como la compatibilidad con la entrada binaria de varias partes al cargar archivos al servidor.
  9. Los desarrolladores prefieren POST para enviar datos de formulario.
